1. Rightmost Smaller Element Distance

You are given an integer array nums of length n.

For each index i, find the largest index j such that:
- j > i
- nums[j] < nums[i]

If such an index exists, set answer[i] = j - i.
Otherwise, set answer[i] = 0.

Return the array answer.

Example 1:
Input: nums = [8,2,5,3]
Output: [3,0,1,0]

Explanation:
- For i = 0, nums[0] = 8. The rightmost index j > 0 with nums[j] < 8 is j = 3, so answer[0] = 3.
- For i = 1, nums[1] = 2. There is no index j > 1 with nums[j] < 2, so answer[1] = 0.
- For i = 2, nums[2] = 5. The rightmost index j > 2 with nums[j] < 5 is j = 3, so answer[2] = 1.
- For i = 3, there is no index to the right, so answer[3] = 0.

Constraints:
- 1 <= n <= 10^5
- -10^9 <= nums[i] <= 10^9

Security Breaching

You are given two integer arrays, layers and energy, each of length n, and an integer k.

An attacker may start at any level i (0 <= i < n) with initial energy k and then move only to the right through levels i, i + 1, ..., n - 1.

For each level j:
1. The attacker must spend layers[j] energy to breach that level.
2. If the attacker does not have enough energy to pay this cost, the attack stops immediately.
3. After breaching level j, if the remaining energy is at least energy[j], the attacker earns 1 point.

For every starting index i, determine the maximum number of points the attacker can collect by starting at level i.

Return an integer array answer of length n where answer[i] is the number of points obtained when starting from level i.

Example 1:
Input: layers = [5, 8, 1], energy = [4, 2, 1], k = 10
Output: [1, 2, 1]

Explanation:
- Start at level 0:
  Spend 5, remaining energy = 5.
  Since 5 >= 4, gain 1 point.
  Next level needs 8 energy, which is not possible, so stop.
  Result = 1

- Start at level 1:
  Spend 8, remaining energy = 2.
  Since 2 >= 2, gain 1 point.
  Move to level 2: spend 1, remaining energy = 1.
  Since 1 >= 1, gain 1 point.
  Result = 2

- Start at level 2:
  Spend 1, remaining energy = 9.
  Since 9 >= 1, gain 1 point.
  Result = 1

Constraints:
- 1 <= n <= 10^5
- 1 <= layers[i], energy[i] <= 10^9
- 1 <= k <= 10^15

You are given a list of retailer locations on a 2D grid.

The i-th retailer is located at coordinates (x_i, y_i). That retailer can deliver to every city inside or on the boundary of the axis-aligned rectangle with corners:
(0, 0), (x_i, 0), (0, y_i), and (x_i, y_i).

In other words, a retailer at (x_i, y_i) can deliver to a city (a, b) if:
0 <= a <= x_i and 0 <= b <= y_i.

You are also given a list of requests, where each request[j] = [a_j, b_j] represents a city.

For each request, return the number of retailers that can deliver to that city.

Note:
- No two retailers are located at the same city.

Example 1:
Input:
retailers = [[1,2], [2,3], [1,1]]
requests = [[1,1], [1,4]]

Output:
[3,0]

Explanation:
- For request [1,1]:
  All three retailers can deliver to (1,1).
- For request [1,4]:
  No retailer has y >= 4, so no one can deliver there.

Example 2:
Input:
retailers = [[3,3], [5,2], [2,5]]
requests = [[2,2], [4,2], [3,4]]

Output:
[3,1,0]

Explanation:
- [2,2] is covered by all 3 retailers.
- [4,2] is covered only by retailer [5,2].
- [3,4] is not covered by any retailer.

Function signature:
vector<int> countRetailers(vector<vector<int>>& retailers, vector<vector<int>>& requests);
